### 示例 - 去除 Chrome 自动填充黄色背景，方法三
```html
<!-- 附上第一幅图片效果的代码 -->
<input
  type="username"
  placeholder="用户名"
  id="username"
  name="username"
  class="login-input"
>

<style>
.login-input {
    background: transparent content-box;
    box-sizing: border-box;
    width: 90%;
    height: 0;
    padding: 15px 5%;
    outline: none;
    border: none;
    border-bottom: 1px solid #333;
    margin-bottom: 20px;
}
</style>
```
